2 - Explique o que são e quais são os Operadores Relacionais. Dê um exemplo do seu funcionamento com um trecho de código

Resposta

Os Operadores Relacionais criam uma relação de comparação entre valores, variáveis e/ou expressões resultando em um valor lógico. Eles são: > (maior que), < (menor que), >= (maior ou igual a), <= (menor ou igual a), == (igual a) e != (diferente de).

a = 2
b = 3
c = 5

c >= a + b -> TRUE
